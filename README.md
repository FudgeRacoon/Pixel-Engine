# Pixel Engine

A graphics library made in `c++` and `SDL2`. 

The library is capable of the following:
  - Drawing a pixels and lines on the screen
  
  - Drawing 2D primitive shapes
    - Rectangles
    - Triangles
    - Circles
    - Polygons
    
 - Rendering 3D meshes
 
 - Loading .obj files
